Modelo SEIR Estendido com Compartimentos Clínicos

Este simulador implementa um modelo SEIR (Suscetível–Exposto–Infectado–Removido) estendido, com compartimentos adicionais para capturar a dinâmica clínica da doença: Hospitalizados (H), Recuperados (Rec) e Fatalidades (F). A integração numérica utiliza o método de Euler com passo temporal dt = 0,1 dia.

1. Fluxo de Compartimentos

S → E → I →
H → F Rec

2. Equações Diferenciais

$$\frac{dS}{dt} = -\beta \frac{S \cdot I}{N}$$ $$\frac{dE}{dt} = \beta \frac{S \cdot I}{N} - \sigma E$$ $$\frac{dI}{dt} = \sigma E - \gamma I$$ $$\frac{dH}{dt} = \tau \gamma I - \alpha H$$ $$\frac{dF}{dt} = \delta_m \cdot \alpha H$$ $$\frac{dR_{ec}}{dt} = (1-\delta_m)\cdot\alpha H + (1-\tau)\gamma I$$

3. Parâmetros do Modelo

SímboloDescriçãoDerivação
$\beta$Taxa de transmissão$R_t / T_{inf}$
$R_t$Número reprodutivo efetivo$(1-\theta)\times R_0$
$\sigma$Taxa de progressão da incubação$1/T_{inc}$
$\gamma$Taxa de remoção (infecciosa)$1/T_{inf}$
$\tau$Fração hospitalizadaparâmetro direto
$\alpha$Taxa de saída hospitalar$1/T_{hosp}$
$\delta_m$Letalidade hospitalar$\delta / 100$
$\theta$Eficácia das NPIsparâmetro direto (0–1)

4. Condições Iniciais

$$S(0) = N - I_0, \quad E(0) = 0, \quad I(0) = I_0$$ $$H(0) = 0, \quad F(0) = 0, \quad R_{ec}(0) = 0$$

5. Integração Numérica (Método de Euler)

$$X(t + dt) = X(t) + \frac{dX}{dt}\bigg|_t \cdot dt, \quad dt = 0{,}1 \text{ dia}$$

Todos os compartimentos são limitados inferiormente a zero: max(0, X), garantindo coerência biológica durante toda a simulação.
